Setup
====

cygwin

    http://www.cygwin.com/

apt-cyg

    http://code.google.com/p/apt-cyg/

ruby

    apt-cyg install ruby

rubygems

    http://rubygems.org/pages/download

    export RUBYOPT=rubygems

	bash < <(curl -k https://rvm.beginrescueend.com/install/rvm)

    gem install bundler
